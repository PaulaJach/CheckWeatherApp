# CheckWeatherApp
In this application you can check current weather in different cities on the world.

## Technologies used in project:
* javaScript,
* flexbox,
* fetchAPI,
* Materialize CSS Framework
* webpack

### Install dependencies
```bash
$ yarn
```

### Build
```bash
$ yarn build
```

All builded files will be in `dist` directory.

### Run in dev mode
```bash
$ yarn dev
```
